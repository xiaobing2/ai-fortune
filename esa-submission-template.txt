作品说明（可直接复制到天池提交结果处使用，可按需微调）：

一、项目简介
本项目是一个基于 Vue 3 + Vite 构建的“AI 算命 / 运势解读”应用。用户输入昵称、生日、星座、当前心情及最关心的问题后，前端在浏览器本地根据规则与伪随机权重生成整体运势、细分维度解读以及行动建议。所有推理在本地完成，不依赖后端服务，也不上传隐私数据，适合部署在阿里云 ESA Pages 上作为轻量 AIGC 边缘应用。

二、创意卓越
1. 将传统“算命/运势”与 AIGC 结合，用更温柔理性的语气重写东方玄学叙事，强调情绪陪伴与自我反思，而非绝对预言。
2. 通过昵称、生日、星座、心情与问题参与生成，使结果兼具“个性化”和“可复现”，避免完全随机带来的割裂感。
3. 界面采用星空/占星氛围的深色渐变背景与发光元素，控制在纯前端 CSS 范围内实现科幻感，移动端也有良好表现。

三、应用价值
1. 可作为品牌活动页面、社群运营、节日互动（如跨年、情人节、毕业季等）的轻量互动工具，提升留存和分享率。
2. 前端完全静态、零后端依赖，部署在 ESA Pages 即可使用，成本低、可复制性强，便于运营团队快速复用或二次改造（例如改成“每日签”、“人生建议卡片”等）。
3. 文案生成逻辑通过多维度文案池组合而成，只需替换文案池内容，就能适配不同业务场景（心理关怀、校园活动、品牌人格测试等）。

四、技术探索
1. 使用 Vue 3 组合式 API 实现一个“种子驱动的规则 AIGC 引擎”，通过 hash 和 pick 机制从多个文案池中选择内容，实现“同样输入 ⇒ 稳定输出 / 不同输入 ⇒ 有差异”的效果。
2. 所有逻辑仅依赖浏览器运行环境，构建产物为纯静态资源，天然与 ESA Pages、边缘缓存兼容，是典型的“边缘原生”小应用形态。
3. 当前实现为本地规则引擎，未来可以进一步在 ESA 边缘函数中接入云端大模型，对部分维度进行补充生成，形成“本地规则 + 边缘 LLM”的混合架构。

五、ESA 使用说明（可根据实际使用情况补充）
1. 使用 ESA Pages 从 GitHub 仓库自动构建和部署项目，构建命令为“npm install && npm run build”，输出目录为“dist”。
2. 可在 ESA 控制台中开启边缘缓存，对静态资源和首页进行缓存，以提升访问时延和并发能力。
3. 后续如接入 ESA 边缘函数，可将表单提交部分改为调用边缘函数接口，进一步做日志记录或高级 AIGC 生成能力。
