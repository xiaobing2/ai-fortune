## 灵象 AI 算命馆 - AIGC 运势解读应用

**本项目由阿里云ESA提供加速、计算和保护。**

> 说明：本项目用于参加“阿里云边缘安全加速ESA · Pages 边缘应用开发大赛”，展示一个基于浏览器端 AIGC 规则引擎的轻量级算命/运势解读应用。

---

### 一、项目介绍

`灵象 AI 算命馆` 是一个基于 **Vue 3 + Vite** 构建的实验性 AIGC 运势解读应用。用户输入昵称、生日、星座、当前心情及最关心的问题后，前端在浏览器本地通过一套“规则 + 伪随机权重”的方式生成：

- 整体运势评价与摘要；
- 感情、事业/学业、财富、健康等维度的文案解读；
- 当日行动建议、幸运色、幸运数字与幸运方向。

所有“推演”均在前端本地完成，不依赖后端服务，也不上传用户隐私数据，非常适合在 **ESA Pages** 上以静态站+边缘缓存的形式部署。

> 使用须知：本项目仅供娱乐与情绪陪伴，不构成任何专业占卜、投资、医疗或法律建议。

---

### 二、功能与交互设计

- **信息输入**
  - **昵称**：用于增强文案代入感。
  - **生日**：参与内部种子生成，使结果对同一人相对“稳定”。
  - **星座（可选）**：预留给后续更精细规则使用。
  - **当前心情**：影响摘要开头的人称与语气。
  - **提问内容**：用于构造种子与文案中的引用，体现“针对性”。

- **AI 运势生成**
  - 使用 `hashString + pick` 组合，将用户输入映射到多个文案池中，生成看似“有性格”的运势文本。
  - 输出包括整体运势等级（如：平衡、小吉、偏稳等）、各维度解读和具体行动建议。
  - 通过幸运色、数字与方向等元素增强娱乐性和可玩性。

- **界面体验**
  - 采用“星空/占星”氛围的深色渐变背景与发光元素，突出神秘感与科技感并存。
  - 表单与结果排版在左右两块卡片中，移动端会自动折叠为上下布局。
  - 清晰标注“娱乐为主”“不作决策依据”等提醒，弱化“玄学绝对权威”的误解。

### 三、技术实现与架构

- **技术栈**
  - **前端框架**：Vue 3（`<script setup>`）
  - **构建工具**：Vite
  - **状态管理**：使用组合式 API + `reactive`/`ref`，不引入额外状态库
  - **样式**：自定义 CSS（不依赖 UI 框架，便于边缘环境下控制体积）

- **核心逻辑**
  - `hashString(str)`：将姓名、生日、星座、心情与问题合并后生成一个稳定的无符号整数种子。
  - `pick(arr, seed, salt)`：基于种子和不同的 salt 从文案池中“抽签”，保证“同一输入 ⇒ 稳定输出”、“不同输入 ⇒ 有差异”。
  - 多个文案池分别对应不同维度（感情、事业、财富、健康、行动建议），形成组合式 AIGC 效果。

- **隐私与安全**
  - 所有计算在浏览器端完成，没有实际网络请求。
  - 不做本地持久化存储（无 localStorage 持久化），关闭页面即清空。

### 四、本地开发与构建

#### 1. 安装依赖

```bash
npm install
```

#### 2. 本地开发

```bash
npm run dev
```

默认访问地址为 `http://localhost:5173`（以终端输出为准）。

#### 3. 生产构建

```bash
npm run build
```

构建输出目录为 `dist/`，可直接上传至阿里云 ESA Pages。

---

### 五、在阿里云 ESA Pages 上部署

1. **准备 GitHub 仓库**
   - 将本项目推送至 GitHub 公开仓库（例如：`https://github.com/your-name/ai-fortune-esa`）。

2. **在 ESA Pages 创建站点**
   - 打开阿里云控制台，进入 **边缘安全加速 ESA → Pages**。
   - 创建新站点，选择“从 Git 仓库导入”，绑定你的 GitHub 账户并选择本仓库。

3. **配置构建参数**
   - **构建命令**：`npm install && npm run build`
   - **产物目录**：`dist`
   - **Node 版本**：推荐选择 Node 20.19+ 或 ESA 默认 LTS，以避免 Vite 的 engine warning。

4. **域名与访问**
   - 部署完成后，ESA Pages 会生成一个官方二级域名作为访问 URL。
   - 如需绑定自定义域名，可在控制台配置 DNS 与 HTTPS。

5. **（可选）结合 ESA 边缘能力**
   - 使用边缘缓存提升静态资源与首页加载速度。
   - 后续可以引入边缘函数，为部分请求增加“云端大模型补充生成”，形成“本地规则 + 云端 LLM”混合模式。

### 六、作品说明（评审维度，可用于提交）

1. **创意卓越**
   - 将传统“算命/运势”与 AIGC 结合，用更温柔理性的语气重写东方玄学叙事，视觉上采用星云式 UI 营造科技占卜氛围。
   - 通过昵称、生日、星座、心情与提问内容参与生成，使结果兼具“个性化”和“可复现”，避免完全随机带来的违和感。
   - 强调“情绪陪伴”和“自我反思”而非绝对预言，让 AI 算命变成一次轻量的内心对话。

2. **应用价值**
   - 作为轻量 H5/小站，适合集成到品牌活动页面、社群运营、节日营销等场景，用于提升用户互动与停留时长。
   - 纯前端、零后端依赖，部署到 ESA Pages 即可使用，成本低、扩散性强，适合运营侧快速复制或二次定制。
   - 可以作为多个行业的“互动组件模板”（不限于算命），简单替换文案池即可演变成每日打卡、心情卡片、随机挑战等玩法。

3. **技术探索**
   - 在浏览器内构建了一个“种子驱动的规则 AIGC 引擎”，用有限的文案池组合出高维度的结果，探索“轻量 AIGC”在边缘应用中的形态。
   - 通过 `hashString + pick` 机制保证结果的一致性和个性化，为后续扩展到 ESA 边缘函数调用云端大模型预留接口思路。
   - 项目完全静态无后端，可与 ESA Pages、边缘缓存以及未来的边缘函数/存储自然结合，成为边缘原生 AIGC 小应用的基础蓝本。
